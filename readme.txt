== Description ==

The halloween blogroll is a Wordpress Plugin/Widget which just gives your blogroll a halloween theme

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget using the page under Appearance -> Widgets
4. Enter the API key in the widget settings
